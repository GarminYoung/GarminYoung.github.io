LCD1602模块（带PCF8574T转接板）

LCD1602模块（带PCF8574T转接板）

参数

显示容量为16 * 2个字符
VCC 模块供电5V
GND 接地
SDA 数据接口
SCL 控制接口

接线

SDA--A4
SCL--A5

代码

#include <Wire.h>
#include <LiquidCrystal_I2C.h> //引用I2C库

LiquidCrystal_I2C mylcd(0x3f,16,2); //创捷液晶对象，设备地址为0x3f(一般为0x27或0x3f），16列2行

void setup(){
  mylcd.init(); //初始化液晶屏
  mylcd.backlight(); //打开液晶屏背光
}

void loop(){
  mylcd.setCursor(5-1, 2-1); //设置光标位置为第2行第5列
  mylcd.print("1"); //输出字符 1
  delay(1000); //延时1秒
  mylcd.clear(); //清屏
  mylcd.setCursor(0, 0); //设置光标位置为第1行
  mylcd.print("2"); //输出字符 2
  mylcd.setCursor(0, 1); //设置光标位置为第2行
  mylcd.print("3"); //输出字符 3
  delay(2000); //延时2秒
  mylcd.clear(); //清屏

}

功能

LCD1602液晶屏在第二行第五列显示“1”，1秒后清屏并在第一行显示”2”，在第二行显示”3”，2秒后清屏.
