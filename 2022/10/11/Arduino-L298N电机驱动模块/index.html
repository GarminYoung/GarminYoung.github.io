L298N电机驱动模块

L298N电机驱动模块

参数

VCC 模块供电12V
GND 接地
5V 模块对外输出5V，当板载5V使能端断开时为模块输入
ENA1 控制电机输出A
IN1 IN2:逻辑输入.
OUT1:电机输出

ENA IN1 IN2 直流电机状态
0 0 0 停止
1 0 0 制动
1 0 1 正转
1 1 0 反转
1 1 1 制动

接线

全速输出

IN1–2
IN2–3

PWN调速输出

IN1–2
IN2–3
ENA1–5
