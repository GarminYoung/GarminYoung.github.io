Microsoft Office

Microsoft Office是由Microsoft(微软)公司开发的一套办公软件套装。
常用组件有Word、Excel、PowerPoint等。

由于其功能的强大和普适性，我们的电脑少不了这个软件。
接下来讲讲如何获得这个软件。

支持正版

office版本

Office 2021是一次性购买的产品，附带适用于 PC 或 Mac 的 Word、Excel 和 PowerPoint 等经典应用，不包括 Microsoft 365 订阅随附的任何服务。一次性购买产品不含升级选项，因此如果你计划升级到下一个主要发行版本，则必须全价购买。

Office Visio是Office软件系列中的负责绘制流程图和示意图的软件，是一款便于IT和商务人员就复杂信息、系统和流程进行可视化处理、分析和交流的软件。

**Microsoft Project(MSPROJ)**是一个国际上享有盛誉的通用的项目管理工具软件，凝集了许多成熟的项目管理现代理论和方法，可以帮助项目管理者实现时间、资源、成本计划、控制。

Microsoft 365是一种订阅，附带 Word、Excel、PowerPoint、OneNote、Outlook、Publisher 和 Access 等高级应用（Publisher 和 Access 仅在 Windows PC 上可用）。这些应用可以安装在多个设备上，包括 PC、Mac、iPad、iPhone、Android 平板电脑和 Android 手机。Microsoft 365 还附带了其他服务，如 1 TB 的 OneDrive 存储空间以及 Microsoft 聊天和电话支持。通过订阅，你可以获取新版本的应用，并会在有更新时自动接收更新。

购买笔记本

正规品牌的笔记本电脑都会自带家庭和学生版，是永久激活的，商品详情页通常会标明这些信息。
激活电脑操作系统和订购office的价钱已经包含在了商品价格之中。

通常情况下，你新购买的笔记本已经安装了office，在规定时间内激活，绑定微软账号即可使用了。

倘若你的电脑上没有安装office，那么你需要：
进入微软官网,登录你的微软账户
右上角头像-我的 Microsoft 账户-订阅-所有订阅
找到“PC 版 Office 家庭和学生版”，点击右边的“查看产品密钥并安装”
点击安装，再点击安装，会下载一个“OfficeSetup.exe”
然后双击运行“OfficeSetup.exe”即可

购买正版

前往Microsoft微软官网购买正版即可。

破解介绍

账号卡密

在某宝平台购买的账号卡密通常分为两种。
第一种是教育版，通过教育edu邮箱免费申请，是一些非常规注册部门的子账号，且子账号受主账号的控制，随时会被删号，即使没有，不久后也会被官方封号。
绑定账号被封后，域名上永久黑名单，不再能订阅购买使用office相关产品。
第二种是由微软非正常渠道流出，订阅价格较官网实惠，但是并不稳定，不建议使用。

激活码

根据微软的机制，一个产品的激活码可以激活多台设备，商家们通过激活次数的超额来实现盈利。
没有安全隐患，但并不属于正版。

KMS

Key management server(KMS)是密钥管理服务，允许在本地网络上激活产品。
这样，单台计算机不必连接至 Microsoft 便可激活产品。需要将一台计算机配置为 KMS 主机。管理员必须为 KMS 主机输入 KMS 主机密钥。必须激活 KMS 主机密钥并连接至 Microsoft 托管的服务器一次，才能激活。
默认情况下，已将 KMS 客户端密钥安装在 Microsoft Office 程序的批量版本中。通过使用 KMS 客户端密钥运行 Microsoft Office 批量版本的计算机称为 KMS 客户端，这些计算机连接至组织的 KMS 主机，以请求激活。最终用户不必执行任何操作，而您也不必为 KMS 客户端输入产品密钥。系统管理员必须通过 Microsoft 激活 KMS 主机并配置 KMS 主机。也可以为 KMS 客户端配置可变时间长度的重新激活和续订期限。
Microsoft Office KMS 客户端通过连接至基于客户端–服务器拓扑的 KMS 主机激活。KMS 客户端可通过使用域名系统 (DNS) 或通过在注册表中设置 KMS 主机名称，来找到 KMS 主机。KMS 客户端使用远程过程调用 (RPC) 与主机联系。最终用户看不到激活过程，激活过程无需任何交互。

大部分KMS存在木马或病毒,容易被他人窃取信息。

激活命令

可以通过激活命令在Office Tool Plus中激活office
Microsoft 365
ospp /insLicID MondoVolume /sethst:kms.loli.beer /setprt:1688 /act

Office LTSC 2021
ospp /insLicID ProPlus2021Volume /sethst:kms.loli.beer /setprt:1688 /act

Office 2019
ospp /insLicID ProPlus2019Volume /sethst:kms.loli.beer /setprt:1688 /act
